# SenSante

Assistant de pre-diagnostic medical pour le Senegal

## Description

SenSante utilise le Machine Learning pour aider au pre-diagnostic des maladies
courantes (paludisme, grippe, typhoide) a partir des symptomes du patient.

## Structure du projet

```txt
.
├── api/ : API FastAPI
├── data/ : Donnees patients (CSV)
├── frontend/ : Interface web
├── models/ : Modele ML sérialisé
├── notebooks/ : Scripts d'exploration

```
